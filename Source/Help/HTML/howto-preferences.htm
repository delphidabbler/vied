How do I ... Save my editing preferences

Editing preferences can be saved as follows:

  1. Create a new document.
  2. Edit the document to to reflect all the default values you wish new documents to have. Leave any string information that varies between documents blank.
  3. If required set or edit the identifier.
  4. Set any comments to be used in the version information file.
  5. Set any comments to be written to any exported resource source file.
  6. Save the preferences using the File | Save As Preferences menu option.
  7. Discard the document without saving unless you want to go on to use it to create a new version information file.

These preferences will be used in all new documents until you either save different preferences or you revert to the program's default values by using the File | Clear Preferences menu option.
