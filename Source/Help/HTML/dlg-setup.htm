User Setup Dialogue Box

This dialogue box allows the user to specify the state in which the program starts up. It is accessed using the Options | User Setup menu option.

The options are specified by ticking or unticking check boxes, as follows:

  • Automatic validation – sets the initial state of the Options | Automatic Validation menu item.
  • Describe the file flags in main windows – sets the initial state of the Options | Describe File Flags menu option.
  • Use UTF-8 as the default encoding for .vi files – ticking the box causes UTF-8 to be used as the default encoding for new projects while clearing the box causes the system default ANSI encoding to be used.

Click OK to accept any changes.

Note: The Automatic validation and Describe the file flags in main windows settings take effect the next time the program is started. The Use UTF-8 as the default encoding for .vi files setting takes place the next time a new project is started: it doesn't affect the current project.
