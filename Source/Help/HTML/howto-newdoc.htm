How do I ... Create a new document

A new document is automatically opened when the program first starts. To discard an existing document and begin a new one simply select the File | New menu option then edit the document as usual.

The new document takes default values from any preferences that have been saved. If there are no preferences then system default values are used.

If the existing document has unsaved changes when File | New is selected then a dialogue box appears that gives the option of saving the changes, loosing the changes or cancelling the creation of the new document.
