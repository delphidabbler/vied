Comments Editor

This dialogue box appears when the user chooses the Edit | RC Comments or Edit | VI Comments menu options.

The dialogue box displays any existing comments of the selected type in a multi-line edit box. The existing comments can be either edited, replaced or deleted. Comments can be multi-line. Pressing return in the dialogue box enters a new line in the edit box - it doesn't close the dialogue box. Click OK to accept the changes.

You can resize this dialogue as required.

Note: Comments will appear at the head of a resource source file delimited by /* and */ symbols or in a version information file preceded by a ';' symbol. These symbols should not be included in the text entered in the dialogue box. The text is not validated.
