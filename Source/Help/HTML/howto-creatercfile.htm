How do I ... Create a resource source file

You can generate a resource source file from the current document by selecting the File | Export menu option. This displays a standard Windows save dialogue box. Select the Resource source files (*.rc) file type, enter a name for the exported file and click OK.

The resulting .rc file contains just a single VERSIONINFO statement, along with any comments you have specified. It can be compiled into a binary resource file using a resource file compiler.

If you have an existing resource file containing more than just version information you can insert the version information into the file.
