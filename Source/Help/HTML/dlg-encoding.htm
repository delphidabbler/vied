Choose .vi File Encoding Dialogue Box

This dialogue box appears whenever you save a .vi file. It appears after you have chosen the file name and after any file overwrite permission is granted.

It allows you to choose which encoding is used when the file is saved. There are two encoding options, chosen by selecting either the ANSI or UTF-8 radio buttons.

The radio button that is selected by default when the dialogue box opens is the same as the encoding noted in the main window caption.

Click OK to accept your choice. The file will be written with the chosen encoding. Clicking Cancel causes the save operation to be cancelled.
