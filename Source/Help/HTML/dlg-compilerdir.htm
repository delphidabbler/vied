Choose Compiler Output Folder Dialogue Box

This dialogue box is displayed when the Edit | Compiler Output Folder menu item is selected.

Specify the name of a folder to be used to receive resource files generated when the current document is compiled. The browse button can be used to select a folder or the folder's name can be entered in the edit control. Either an absolute path or a path relative to the .vi file location can be specified.

Click OK to close the dialogue box and accept the folder.
