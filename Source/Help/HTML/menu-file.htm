File Menu

New (Ctrl+N) Creates a new, un-named, document in memory and displays it. The contents of the new document reflect the user's current preferences. If there are any unsaved changes in the current document you will be prompted to save changes before continuing.
Open (Ctrl+O) Opens an existing version information file using the Open File dialogue box. If there are any unsaved changes in the current document you will be prompted to save changes before continuing.
Save (Ctrl+S) Saves the current document to disk as a version information file. If the document has not yet been saved then the process is the same as that described for the Save As menu option below.
Save As (Ctrl+A) Permits the current document to be saved as a version information file under a new name. The Save File dialogue box is displayed where a new name and location for the file can be chosen. Following that the Choose .vi File Encoding Dialogue Box appears where you select the text encoding to use when writing the file. Options are the system defualt ANSI encoding and UTF-8 encoding. The file is then saved.
Export (Ctrl+E) Enables a resource file to be created from the current document. Both resource source (.rc) files and binary resource files (.res) can be created. The .res file type is only available if a resource compiler has been registered with the program. The Export File dialogue box is displayed where the name and type of the exported file can be specified.
Compile (F9) Causes the current document to be compiled into a binary resource file. This option is disabled unless the program has been configured to work with an external resource compiler. If a default output folder for the compiler has been specified for this document, and the document has been saved, the output file has the same name as the document and is stored in the designated folder. Otherwise a dialogue box is displayed where the name of the .res file can be entered. To specify a default output folder use the Edit | Compiler Output Folder menu option.
Save As Preferences Causes the current document to be saved as preferences. These preferences are used as the default whenever a new document is created, either at program start-up or when the File | New menu option is selected.
Clear Preferences Causes any saved preferences to be cleared. After preferences are cleared any new documents will have built-in default values.
View RC Statements Displays the View RC Statements dialogue box which shows how the current document will appear when converted to a VERSIONINFO resource statement
View Macro Values Displays the View Macro Values dialogue box which displays any currently defined macros and their values.
Exit Exits the program. If there are any unsaved changes to the current document you will be prompted to save the changes before closing.
