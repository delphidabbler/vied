Save File Dialogue Box

This standard Windows dialogue box appears when the user chooses the File | Save As menu option, or File | Save when the file has not yet been saved.

It allows the user to choose file in which to save the current document as a version information file. The file must have a .vi extension. After saving the file name appears in the window caption and will be used each time the File | Save menu option is chosen.

If the .vi extension is not used an error message is displayed and the file is not saved.
