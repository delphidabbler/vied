Edit Menu

Edit Current Item (F2) Displays a dialogue box where the currently selected item in the main display can be edited. The form the dialogue box takes depends on the selected item. See How to Edit an Item for more information.
Clear Current Item (Ctrl+Del) Clears the value of the currently selected item in the main display. When a value is cleared the action depends on the value. Fixed File Information and Translation Information items are reset to their system default values. String Information values are simply deleted.
RC Identifier Displays the Identifier Editor dialogue box where the identifier used in the VERSIONINFO resource statement can be edited. Identifiers are not required. To see how any identifier is defined in a resource statement use the File | View RC Statements menu option.
RC Comments Displays the Comments Editor dialogue box where the comments written to exported resource source files may be edited.
VI Comments Displays the Comments Editor dialogue box where the comments written to saved version information files can be edited.
Macros Displays the Macro Editor dialogue box where macros can be created, edited and deleted.
Compiler Output Folder This option displays a dialogue box where the folder used to receive compiled binary resource files can be specified. Compiled files are created when the File | Compile menu option is used. If the program has not been set up to use an external resource compiler then this menu option will be disabled.
Analyse Analyses the current document for validity. Any errors are listed in the Analysis Errors dialogue box.
Copy (Ctrl+C) Copies the information in the current document to the clipboard as plain text in resource source file format.
