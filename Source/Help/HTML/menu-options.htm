Options Menu

Automatic Validation Toggles automatic validation on and off. A tick next to the menu option indicates that automatic validation is on. The default state of this option can be specified by using the Options | User Setup menu option.
Describe File Flags Toggles between two methods of displaying File Flags and File Flags Mask information. When the menu option is ticked the information is displayed in full, i.e. all the file flags in a bit set are identified by name. When the option is off just the hexadecimal value of the bit set is displayed. In either case the full description is used when editing these items. The default state of this option can be specified by using the Options | User Setup menu option.
User Setup Displays the User Setup dialogue box where global program options can be modified.
Resource Compiler Displays the Specify Resource Compiler dialogue box which is used to provide details any the external resource compiler used to compile or export binary resource files. To remove any compiler just delete its details in the dialogue box.
