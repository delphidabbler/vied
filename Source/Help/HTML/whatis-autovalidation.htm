What is ... Auto-validation

When automatic validation is switched on invalid information in a version information file is corrected as the file is loaded. This also applies to the preferences used when creating a new document.

Additionally while editing the current document while automatic validation is switched on:

  1. The user is warned when an attempt is made to delete string information that is compulsory.
  2. The "Private Build" and "Special Build" strings can only be edited if the appropriate File Flags are set, otherwise any values assigned to these string items will be deleted.
  3. Inappropriate File Sub-Types are inhibited. (Valid File Sub-Types are dependent on the File Type).

When automatic validation is switched off many of these checks are not carried out.

Automatic validation can be switched on or off using the Options | Automatic Validation menu option. The default setting for this feature can be set in the User Setup dialogue box.
