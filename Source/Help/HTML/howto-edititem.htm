How do I ... Edit a version information item

Version information items are discrete peices of version information, each displayed on a different line in the main editor window.

Each item is edited by double clicking it or selecting it with the mouse or arrow keys an pressing F2 or choosing the Edit | Edit Current Item menu option.

An editor dialogue box appears related to the type of item being edited. The following dialogue boxes are used:

  • Fixed File Info section

    • Version Number Editor: Used to edit the File Version # and Product Version # items.
    • Pick List Editor: Used to edit the File OS and File Type items. It is also used for File Sub-Type when File Type is either VFT_DRV or VFT_FONT.
    • Number Editor: Used to the File Sub-Type item when File Type is set to VFT_VXD.
    • Bit set editor: Used to edit the bit sets used for the File Flags Mask and File Flags items.

  • Translation Info section

    The Pick List Editor is used to edit both the Language and Character Set items.

  • String Info section

    Each string info item is edited using the String Editor. Note that this editor permits the use of fields.

    When automatic validation is switched on an error message is displayed if the Private Build or Special Build items are selected, have no values and the appropriate File Flag is not set. In the case where these items have a value a warning appears giving the option of deleting the entry.

To delete an item select it the press Ctrl+Del or select the Edit | Clear Current Item menu option. When cleared, fixed file and translation information items are reset to their system default value and string information items are cleared.
