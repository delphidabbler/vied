What is ... An Identifier

A VERSIONINFO resource statement begins with a resource identifier which must have value 1. However, this value may be stored in a symbolic identifier.

The user may specify an identifier to use for this purpose by choosing the Edit | RC Identifier menu option. The current preferences store the default identifier (if any).

If an identifier is specified it has to be defined to have value 1 in the resource file. The editor creates a suitable definition if an identifier is specified.

The maximum length of the identifier is 31 characters, it may only contain letters, digits and underscores and it may not begin with a digit.
