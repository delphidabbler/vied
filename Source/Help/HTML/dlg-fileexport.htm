Export File Dialogue Box

This standard Windows dialogue box appears when the user chooses the File | Export menu option. It is used to choose the name, path and type of file to be exported.

Select the required type of file using the drop-down list of file types. Available types are:

  • Resource source files (.rc): the current document is exported in resource source file format with the .rc extension. Use the File | View RC Statements menu option to see what the contents of the file will look like.
  • Resource binary file (.res): the current document is exported in binary resource format with the .res extension, ready for inclusion in an executable file. Note: This file type is only available if a resource compiler has been registered with the program.

If an invalid extension is entered an error message appears and the operation is cancelled.
