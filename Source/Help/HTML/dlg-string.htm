String Information Editor

This dialogue box appears when you edit any STRING INFO item for which editing is currently permitted. You can resize this dialogue box if you need a larger area in which to enter strings.

The current value of the string is displayed in a multi-line edit box at the top of the dialogue box. The string can be edited as desired. Note: Pressing return doesn't enter a new line of text but may close the dialog box. This is because STRING INFO text must be on one logical line.

Fields and Macros can be entered in the text as follows:

  1. Place the cursor where the field/macro is to be entered in the text.
  2. Select the required field/macro from the Insert Field drop-down list. This list contains all valid fields/macros for the STRING INFO item being edited.
  3. Click the Insert button.

Fields and macros can also be entered by typing them directly into the text, enclosing the name by angle brackets. To enter a < character in the text it must be entered as <<>.

Click OK to accept your changes. Any trailing spaces are trimmed from the returned strings

Note: When OK is clicked the entry is validated. If an unknown field or macro is encountered an error message is displayed. Dismissing the error message returns you to the dialogue box with the erroneous field/macro highlighted ready for correction. If automatic validation is switched on and no text (ignoring spaces) has been entered for a required STRING INFO item then a warning is displayed with the option of ignoring the warning or returning to the dialogue box to enter some text.
