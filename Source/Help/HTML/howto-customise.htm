How do I ... Customise the display

Only one aspect of the main display can be customised, and that is how the File Flags and File Flags Mask bitsets are displayed. They can be displayed in one of two ways:

  • using the symbolic constants for all the flags in the bit set
  • the hexadecimal value of the bitset

Which option to use is controlled by the Options | Describe File Flags menu option. The the display state used when the application starts is specified in the User Setup dialogue box accessed from the Options | User Setup menu option.
