How do I ... Compile a binary resource file

The current document can be compiled to a binary resource file by choosing the File | Compile menu option.

If a default output folder for the compiler has been specified for this document, and the document has been saved, the output file has the same name as the document and is stored in the designated folder. Otherwise a dialogue box is displayed where the name of the .res file can be entered. To specify a default output folder use the Edit | Compiler Output Folder menu option.

Note: A resource compiler must have been registered with this program before this facility can be used.
