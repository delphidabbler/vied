View Macros Values Dialogue Box

This dialogue box is displayed when the user chooses the File | View Macro Values menu option.

The dialogue displays either one or two tabs: the Macros tab and the Warnings tab, which is only displayed if there are warnings.

All the currently defined macros are displayed in the Macros tab, along with their values. The macros are displayed in a table.

Any externally referenced files that can't be found are noted in the Warnings tab.
